# LOG2E

`MathPlus.LOG2E` 속성은 밑이 2인 로그 $e$를 나타내며, 약 1.442의 값을 가집니다.

$$
\log_{2} e \approx 1.442
$$

```javascript
import MathPlus from '@ony3000/math-plus';

console.log(MathPlus.LOG2E);
// expected output: 1.4426950408889634
```
