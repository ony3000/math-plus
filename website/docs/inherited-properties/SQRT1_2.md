# SQRT1_2

`MathPlus.SQRT1_2` 속성은 2의 제곱근의 역수를 나타내며, 약 0.707의 값을 가집니다.

$$
\frac{1}{\sqrt 2} = \frac{\sqrt 2}{2} \approx 0.707
$$

```javascript
import MathPlus from '@ony3000/math-plus';

console.log(MathPlus.SQRT1_2);
// expected output: 0.7071067811865476
```

---
