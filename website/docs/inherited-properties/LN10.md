# LN10

`MathPlus.LN10` 속성은 10의 자연로그 값을 나타내며, 약 2.302의 값을 가집니다.

$$
\ln 10 \approx 2.302
$$

```javascript
import MathPlus from '@ony3000/math-plus';

console.log(MathPlus.LN10);
// expected output: 2.302585092994046
```

---
