# LOG10E

`MathPlus.LOG10E` 속성은 밑이 10인 로그 $e$를 나타내며, 약 0.434의 값을 가집니다.

$$
\log_{10} e \approx 0.434
$$

```javascript
import MathPlus from '@ony3000/math-plus';

console.log(MathPlus.LOG10E);
// expected output: 0.4342944819032518
```
