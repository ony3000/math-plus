# SQRT2

`MathPlus.SQRT2` 속성은 2의 제곱근을 나타내며, 약 1.414의 값을 가집니다.

$$
\sqrt 2 \approx 1.414
$$

```javascript
import MathPlus from '@ony3000/math-plus';

console.log(MathPlus.SQRT2);
// expected output: 1.4142135623730951
```
