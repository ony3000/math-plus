# E

`MathPlus.E` 속성은 자연로그의 밑 $e$를 나타내며, 약 2.718의 값을 가집니다.

$$
e = \lim_{n \to \infin} \left( 1 + \frac{1}{n} \right)^{n} \approx 2.718
$$

```javascript
import MathPlus from '@ony3000/math-plus';

console.log(MathPlus.E);
// expected output: 2.718281828459045
```
